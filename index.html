This project is a simple mono-alphabetic cipher. In the Key field, a number can be entered. Plain text can then be entered into the text-area. Once submitted, an enciphered version is returned on the right side of the page.

Enciphering is a simple process of creating a cipher alphabet by shifting the standard alphabet to the left the amount of places equal to the key. For example, a key of '3' will create a cipher alphabet that will be shifted 3 positions to the left of the standard alphabet. So, if a plaintext "d" is submitted, the enciphered text will return an "a".

Future plans are to create a poly-alphabetic cipher, and increment the page until a replica Enigma machine has been created.
